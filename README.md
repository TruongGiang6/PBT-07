# PBT-07 - JAVASCRIPT BASICS — Variables, Data Types, Control Structures

Dự án này chứa các bài tập thực hành và lý thuyết nền tảng về ngôn ngữ lập trình JavaScript, bao gồm cách khai báo biến, cấu trúc điều khiển, vòng lặp và xử lý dữ liệu cơ bản.

## 📁 Cấu trúc thư mục

```text
PBT-07/
├── screenshots/
├── answers.md
├── calculator.js
├── fizzbuzz.js
├── guess.js
├── guess_number.html
├── restaurant_bill.js
├── student_data.js
├── var_let_const.js
└── README.md
```

- `screenshots/`: Chứa các hình ảnh minh họa kết quả chạy code trên console hoặc trình duyệt.
- `answers.md`: Câu trả lời cho các câu hỏi lý thuyết liên quan đến JavaScript (ví dụ: phân biệt `var`, `let`, `const`, vòng lặp, v.v.).
- `var_let_const.js`: Bài tập minh họa sự khác nhau về phạm vi (scope) và hoisting giữa `var`, `let`, và `const`.
- `calculator.js`: Viết các hàm thực hiện các phép toán cơ bản (cộng, trừ, nhân, chia).
- `fizzbuzz.js`: Bài toán FizzBuzz kinh điển sử dụng vòng lặp và câu lệnh điều kiện.
- `restaurant_bill.js`: Bài tập tính toán hóa đơn nhà hàng (tính tiền tip, tổng cộng).
- `student_data.js`: Bài tập làm quen với kiểu dữ liệu đối tượng (Object) và mảng (Array) để quản lý thông tin.
- `guess_number.html` & `guess.js`: Bài tập tương tác "Đoán số" kết hợp giữa HTML và JavaScript (nhận đầu vào từ người dùng và xử lý logic trò chơi).
